Introduction¶

Every day, more than 80,000 commercial flights take place around the world, operated by hundreds of airlines. For all aircraft take-off weight exceeding 27 tons, a regulatory constraint requires companies to systematically record and analyse all flight data, for the purpose of improving the safety of flights. Flight Data Monitoring strive to detect and prioritize deviations from standards set by the aircraft manufacturers, the authorities of civil aviation in the country, or even companies themselves. Such deviations, called events, are used to populate a database that enables companies to identify and monitor the risks inherent in these operations.

This notebook is designed to let you manipulate real aeronautical data, provided by the Safran Group. It is divided in two parts: the first part deals with the processing of raw data, i.e. to perform pre-processing and visualizations needed to understand and clean the data. The second part deals with actual data analysis, and covers many interesting problems.

Part 1: Data processing¶

1.1 Loading data¶

Load the RAW data into the structure proposed below (the schema is given to you), using the code provided. The definition of the variables available is given here:

  • T: time (Hz)
  • LATP_1: Latitude ()
  • LONP_1: Longitude ()
  • RALT1: Radio Altitude, sensor 1 (feet)
  • RALT2: Radio Altitude, sensor 2 (feet)
  • RALT3: Radio Altitude, sensor 3 (feet)
  • ALT_STD: Relative Altitude (feet)
  • HEAD: head ()
  • PITCH: pitch ()
  • ROLL: roll ()
  • MACH: Mach (Mach) number
  • IAS: Tentative Air Speed (m/s)
  • N11 : speed N1 of the first engine (% of tr/min)
  • N21 : speed N2 of the second engine (% of tr/min)
  • N12 : speed N1 of the second engine (% of tr/min)
  • N22 : speed N2 of the second engine (% of tr/min)
  • AIR_GROUND: status in the air (Boolean)
  • AP_ENGAGED: active Autopilot mode (Boolean)
  • AP_OFF: Autopilot off (Boolean)

Note: The time unit of variable T is 8Hz. If you want to obtain the time in seconds, you need to divide this varible by 8.

import csv
import numpy as np
from os import listdir
from os.path import isfile, join
BASE_DIR = "/safran/"

#fonction de chargement des donnes
def load_data_from_directory(DATA_PATH):
    files_list = [f for f in listdir(DATA_PATH) if isfile(join(DATA_PATH, f))]
    Fe = 1
    flight_list = list()
    
    print("There are %d files in total" % len(files_list))
    
    # we only load data of the first 390 files
    max_flight = 390
    print("We process %d files" % max_flight)
    
    for ind_file,afile in enumerate(files_list):
        flight_list_struct = {}
        # print the percentages to keep track the process status
        if np.mod(ind_file,max_flight/10)==0:
            print(np.round(float(ind_file)/float(max_flight)*100),'%')
        
        # only process the first "max_flight" files
        if ind_file <max_flight:
            variables_names = list()
            data_dictionary = {}
            mask_dictionary = {}
            file=open(join(DATA_PATH,afile), 'r')
            csv_file=csv.reader(file,delimiter=',')
            for ind_row,row in enumerate(csv_file):
                # extract the header and initialize "columns"
                if ind_row==0:
                    variables_names = row
                    for id_var,var_name in enumerate(variables_names):
                        data_dictionary[var_name]=list()
                        mask_dictionary[var_name]=list()
                else:
                    for id_var,var_name in enumerate(variables_names):
                        try:
                            # get the value at the current column
                            a = row[id_var]
                            # append the value to the corresponding column
                            data_dictionary[var_name].append(float(a))
                            # mark the value is correct
                            mask_dictionary[var_name].append(1)
                        except:
                            # if the value is empty (missing value)
                            if a=='':
                                data_dictionary[var_name].append(-1)
                            # if the value can not be converted to float
                            else:
                                data_dictionary[var_name].append(a)
                            # mark the value is incorrect
                            mask_dictionary[var_name].append(0)
            # to get good performance in processing
            # we convert the list values in each column to a numpy array
            for id_var,var_name in enumerate(variables_names):
                data_dictionary[var_name]=np.array(data_dictionary[var_name])
                mask_dictionary[var_name]=np.array(mask_dictionary[var_name],dtype=bool)

            flight_list_struct['DATA']=data_dictionary
            flight_list_struct['MASK']=mask_dictionary
            file.close()
        else:
            break
        
        # the data of each file is stored in seperated data structures, which are simple list
        flight_list.append(flight_list_struct)
    print('data loaded')
    return flight_list,variables_names
